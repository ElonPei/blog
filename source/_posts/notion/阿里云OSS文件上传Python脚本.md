# **oss2mk**

由于`MWeb`不支持阿里云图床，本脚本的主要功能时把图片上传到阿里云oss并把图片的markdown语法复制到粘贴板。

## **运行环境**

```
 系统          : macOS 10.13
 Python版本    : 3.6.1
```
