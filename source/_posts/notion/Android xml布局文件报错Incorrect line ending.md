## #### 问题描述

> 在写界面布局时，某控件的最后行报如下错误
> 

```
Incorrect line ending: found carriage return (\r) without corresponding newline (\n)
```

## #### 解决方法

> project –> clear （可以解决，但再次保存文件时又报错）ctrl+a 全选 –> ctrl+x剪贴 –> ctrl+s 保存 –> ctrl+v 复制 –> ctrl+s 保存
>
